# Feature To Do List

This list is organized by practical value for dealerships using the app with their existing CRM, cashier, DMS, or repair order software.

- [ ] Add database backup and restore from Settings.
- [ ] Add CSV/Excel import for brands and parts.
- [ ] Add CSV/Excel export for brands and parts.
- [ ] Add bulk edit tools for part numbers, categories, models, and brands.
- [ ] Add audit history for part changes.
- [ ] Add optional employee login.
- [ ] Save favorites by employee account when login is enabled.
- [ ] Add manager/admin roles for brand deletion, backup, import, and bulk edit.
- [ ] Add automatic database backup before brand deletion, permanent removal, imports, or bulk edits.
- [ ] Add a missing part numbers report.
- [ ] Add a recently changed parts report.
- [ ] Add a duplicate part number report.
- [ ] Add a print-friendly part list by brand or department.
